# 🤖 Sistema de Agentes

## O que é um agente

Um agente é uma entidade de IA com:

- Nome
- Objetivo
- Personalidade
- Prompt-base
- Especialização

---

## Estrutura


{
  "name": "Aurora",
  "goal": "Atendimento imobiliário",
  "personality": "Humana e acolhedora"
}

### Funções
	•	Responder usuários
	•	Manter contexto
	•	Evoluir com memória
	•	Trabalhar em equipe


### Tipos de agentes
	•	Atendimento
	•	Jurídico
	•	Estratégico
	•	Técnico
